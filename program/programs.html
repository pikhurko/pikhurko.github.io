Oleg Pikhurko: Some programs


Some of the files below are tar archives compressed by the gzip utility. Please refer for the file README (or the comments in the source) in each distribution for further information. All these programs are distributed under GNU General Public License. Comments, bug reports, etc, will be greatly appreciated.

piano.tgz
defines a callable C function for generation of all piano chords which satisfy the combinatorial properties specified by the user.

plattice.tgz
a program in C for proving lower bounds on the function beta(d,l) defined in the e-print "Lattice Points inside Lattice Polytopes," arXiv:math.CO/0008028

kostochka.tgz
generation of  Kostochka's family of (4,3)-Turan hypergraphs

srbip.c
a program computing the asymptotics of the size Ramsey number of complete bipartite graphs. The description of the algorithm (and the proof of its correctness) can be found in the e-print "Asymptotic Size Ramsey Results for Bipartite Graphs," arXiv:math.CO/0101197
